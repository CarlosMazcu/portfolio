01. Raftel Graphics Engine

Raftel Engine is an advanced graphics engine developed in C++ and based on modern OpenGL. It was created as an academic project in collaboration with Marc Folgado, applying advanced knowledge in graphics programming, systems architecture, and engine design.

It features an optimized rendering system (Forward and Deferred Rendering), HDR lighting, customizable shading, Entity-Component System (ECS), material management, and compatibility with multiple 3D formats.

Additionally, it includes a graphical user interface with ImGui, real-time editor, integration with Lua scripting, multithreading support via JobSystem, basic physics with collisions and gravity simulation, and a modular system for future extensions.
